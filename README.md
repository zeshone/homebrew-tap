# homebrew-tap

Homebrew tap for Zesh-One tools.

## Install

```sh
brew tap zeshone/tap
brew install doc-agent-ai
```

Then:

```sh
doc-agent-ai install
```

## Available formulae

- **doc-agent-ai** — Multi-platform documentation workflow agent installer ([source](https://github.com/zeshone/doc-agent-ai))

## Update

```sh
brew update
brew upgrade doc-agent-ai
```

## Uninstall

```sh
brew uninstall doc-agent-ai
brew untap zeshone/tap
```

---

Formulae in this tap are published automatically by [goreleaser](https://goreleaser.com) on every tagged release of the upstream repo. Do not edit `Formula/*.rb` by hand — changes will be overwritten by the next release.
